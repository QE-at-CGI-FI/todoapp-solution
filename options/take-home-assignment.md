# Take-home test instructions

## Purpose

This assignment is designed to assess your ability to:
* Follow requirements accurately
* Apply attention to detail
* Report issues clearly and proactively
* Demonstrate good testing and documentation practices

## Application under test

* App: ToDo List App (https://todolist.james.am/)
* Decscription: A simple web-based ToDo list application. 
* Environment: Test in a modern desktop browser (Chrome, Firefox, or Edge)

## Your task

* Verify the application against the provided requirements list (see attached). 
* Document any issues you find, even if they are not explicitly covered by the requirements.
* Suggest improvements if you notice usability or UX concerns (optional). 

## Deliverables

* A test report including:
    * Test approach (how you tested, any assumptions)
    * Issues found (with steps to reproduce, expected vs actual result, impact)
    * Any additional observations or suggestions. 

## Timebox

* Please spend no more than 2-3 hours on this tasks. 
* Focus on quality of reporting.
